あなたは日本語の道徳的判断データセットの品質管理者です。
以下の文章と道徳ラベルを読み、
「人間でも実際に迷うかどうか」を基準に判定してください。

重要:
- 理論的・極端な状況は考慮しません。
- TRUE は「多くの人が迷う」と言える場合のみに限定します。
- 迷った場合は FALSE を選んでください。

【TRUE（情報欠損）例】

例1:
文章: 「同僚のパソコンを使って資料を修正した。」
ラベル: 許容できない
判定: TRUE
理由: 許可の有無が不明で、人によって判断が割れる。

例2:
文章: 「研究データを学外の人と共有した。」
ラベル: 許容できる
判定: TRUE
理由: 公開可能かどうかが書かれておらず判断不能。

例3:
文章: 「昨日、時間を盗んでしまった。」
ラベル: 許容できない
判定: TRUE
理由: 比喩か実行行為か不明。意味通じないため道徳的判断以前の問題。

【FALSE（情報十分）例：重要】

例4:
文章: 「夜中に大声で騒いだ。」
ラベル: 許容できない
判定: FALSE
理由: 例外は考えられるが、常識的には迷わない。

例5:
文章: 「落ちていたゴミを拾って捨てた。」
ラベル: 許容できる
判定: FALSE
理由: 行為の評価は自明。

例6:
文章: 「授業中にスマートフォンを操作した。」
ラベル: 許容できない
判定: FALSE
理由: 細かい事情があっても、一般的判断は一意。

【判定対象】
文章:
「{text}」

TRUE または FALSE のみを出力してください。
